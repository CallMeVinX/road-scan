# Placeholder Images untuk Road Scan

Folder ini berisi gambar placeholder untuk demonstrasi aplikasi Road Scan.

## Gambar yang tersedia:

1. road_broken* - Gambar jalan yang memilki kerusakan
2. road_safe* - Gambar jalan yang tidak memiliki kerusakan

## Cara menambahkan gambar:

1. Letakkan file gambar (JPG/PNG) di folder ini
2. Pastikan nama file deskriptif
3. Gambar akan otomatis terdeteksi oleh API

## Format yang didukung:
- JPG
- PNG
- Maksimal ukuran: 10MB per gambar
